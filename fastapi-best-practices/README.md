# FastAPI Best Practices Skill

This skill contains a collection of best practices for building high-performance, scalable FastAPI applications. It is based on real-world production experience and community guidelines.

## Contents

- **Rules**: Individual rule files in `rules/` directory covering various aspects of FastAPI development.
- **Compiled Guide**: `AGENTS.md` contains the full set of rules in a single document.
- **Skill Definition**: `SKILL.md` defines the skill metadata and usage instructions.

## Usage

Agents should refer to this skill when:
- Scaffolding new projects or features.
- Reviewing code for performance or security issues.
- Refactoring existing codebases.
